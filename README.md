# Renan Bazinin Portfolio

Personal portfolio site for [www.renanbazinin.com](https://www.renanbazinin.com), built with React, Vite, React Router, Framer Motion, and Lucide icons.

## Tech Stack

- React 19
- Vite
- React Router
- Framer Motion
- Lucide React

## Local Development

```bash
npm install
npm run dev
```

## Quality Checks

```bash
npm run lint
npm run build
```

## Deployment

The site is configured for GitHub Pages and the custom domain in `CNAME`.

```bash
npm run deploy
```
